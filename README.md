###  Features
- Register a handle (like `@arkwinger`) to a phone number
- Resolve a phone number or handle to find its mapping
- Data persistence with SQLite
- Public REST API powered by FastAPI
- Hosted with:
  - **Frontend:** GitHub Pages  
  - **Backend:** Render  

---

###  Idea
This project explores what a "Phone Name System" could look like ‚Äî a global identity layer that maps phone numbers to human-friendly names, just like DNS maps IPs to domain names.
